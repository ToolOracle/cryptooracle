| Tool | Description |
|------|-------------|
| `crypto_price` | Live cryptocurrency prices in USD/EUR with 24h change, volume, and market cap. S |
| `crypto_detail` | Full coin profile: price, market cap, ATH, supply, 7d/30d change, description, l |
